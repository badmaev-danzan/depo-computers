# Библиотека книг

Приложение для управления личной библиотекой книг на Angular 20 + NgRx.

## Технологии

- Angular 20
- NgRx
- Angular Material
- JSON Server

## Установка и запуск

### 1. Установка зависимостей

```bash
npm install
```

### 2. Запуск приложения

**Вариант 1 - Одной командой:**
```bash
npm run dev
```

**Вариант 2 - В двух терминалах:**

Терминал 1 - Backend (JSON Server):
```bash
npm run api
```

Терминал 2 - Frontend (Angular):
```bash
npm start
```

### 3. Открыть в браузере

```
http://localhost:4200
```
